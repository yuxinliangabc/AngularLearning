第9课学习 指令详解

1.指令scope作用域

直接调用相同的作用域对象 默认 scope = false

从当前作用域对象继承一个新的作用域对象 scope = true

创建一个同当前作用域相隔离的作用域对象 scope = {}

scope的三个用法 【外部作用域】、【继承作用域】、【隔离作用域】

Outside myDirective: {{ myProperty }}

scope = true 相当于子指令创建了自己的新的作用域

Outside myDirective: {{ myProperty }}

scope = false 表明子指令没有自己的作用域，用了和外部外部作用域相同的作用域


Surrounding scope: {{ myProperty }}

scope = {} 表明隔离的作用域，指令内不能使用外部作用域的变量
